Safeguard · Standard Operating Procedure
Receiving &
Releasing Children
What this is for: a door-and-binder reminder for the start and end of class. Use it when children arrive, when parents pick up, and anytime a volunteer is unsure whether a child can be released.
Core rule · no exceptions
Children are released only to authorized adults, with accurate sign-in and sign-out records every time.
Check-in
Receiving
  1. Greet child and parent/guardian.
  2. Confirm child name and class placement.
  3. Parent/guardian signs the child in on the class sheet.
  4. Note allergies, medical needs, or special instructions.
  5. Confirm room ratio before admitting child.
Use SG-FRM-008
Check-out
Releasing
  1. Confirm adult is on the authorized pickup list.
  2. Ask for ID if the adult is unfamiliar.
  3. Parent/guardian signs the child out on the class sheet.
  4. Note late pickup or unusual circumstances.
  5. Leader initials release for accountability.
Use SG-FRM-008
Never release a child when...
  • Adult is not listed by parent/guardian.
  • Adult identity is uncertain.
  • Custody concern or conflict is active.
  • Sign-out record is incomplete.
If uncertainKeep the child with approved volunteers, contact the ministry leader immediately, and record the exception for coordinator review.
